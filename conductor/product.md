# Product Guide: CodeUI

## Initial Concept
CodeUI is a modern, AI-powered code editor interface built with Next.js 16, React 19, and TypeScript. It features a dark theme design with a sidebar for AI chat interactions and a main content area for code editing and preview. The project leverages shadcn/ui components for a consistent and accessible UI experience.

## Target Audience
CodeUI serves a diverse range of users, empowering them to bridge the gap between design and code:
*   **Junior & Professional Developers:** To accelerate workflows, reduce cognitive load, and learn modern styling (Tailwind CSS) through visual-to-code translation.
*   **UI/UX Designers:** To transition seamlessly from static prototypes to functional code without needing to master complex CSS syntax.
*   **Startup Founders & Entrepreneurs:** For rapid MVP building, creating landing pages, and developing interactive prototypes with zero platform lock-in.
*   **Freelancers & Agencies:** To streamline client project delivery using AI-assisted generation and portable single-file exports.
*   **Frontend Developers:** Looking for AI-assisted UI generation.
*   **Software Engineers:** Needing a streamlined environment for building React/Next.js components.

## Core Value Proposition
CodeUI acts as an intelligent intermediary in the development process. By integrating AI chat directly into the editor, it allows users to iterate on designs and code simultaneously. The ability to export single-file components ensures that code created in CodeUI is portable and production-ready, removing the friction often associated with "low-code" or "no-code" tools that impose platform constraints.

## Key Features
*   **AI-Powered Chat Interface:** A sidebar dedicated to AI interactions for generating and refining code.
*   **Integrated Code Editor:** A robust editing environment powered by Monaco Editor.
*   **Live Preview:** Instant visualization of generated UI components.
*   **Surgical Reprompting:** Advanced partial update mechanism using a custom SEARCH/REPLACE protocol, allowing precise AI edits without full-file rewrites.
*   **Modern Stack:** Built on Next.js 16, React 19, and Tailwind CSS 4 for high performance and maintainability.
*   **Portable Exports:** Generate code that can be easily dropped into any React/Next.js project.
*   **Secure Authentication:** Google-based sign-in via NextAuth.js to manage user sessions and protect resources.
